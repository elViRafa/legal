Política de Privacidade - PlantãoPro

Última atualização: 09 de Maio de 2026

A sua privacidade e a segurança dos dados dos seus pacientes são prioridades fundamentais para o PlantãoPro. Esta política descreve como as informações são tratadas no aplicativo.

1. Natureza do Aplicativo (Offline-First)

O PlantãoPro é um aplicativo projetado com a arquitetura offline-first. Isso significa que, por padrão, todas as informações de pacientes, diagnósticos, checklists e evoluções clínicas são armazenadas exclusivamente no armazenamento local do seu dispositivo móvel. Não possuímos servidores de banco de dados para armazenar os dados dos seus pacientes.

2. Dados Coletados

2.1. Dados de Pacientes (Inseridos pelo Usuário)

O aplicativo permite que você insira dados como:

  • Nome do paciente e número do leito;
  • Diagnósticos e setor/enfermaria;
  • Evoluções clínicas, prescrições e pedidos de exames.

Estes dados são seus e permanecem no seu dispositivo. O desenvolvedor não tem acesso a essas informações em nenhum momento.

2.2. Dados de Pagamento e Assinatura

Utilizamos o serviço de terceiros RevenueCat para gerenciar as assinaturas PRO através da Google Play Store. O RevenueCat pode coletar informações anônimas de transação e identificadores de dispositivo para validar sua assinatura, mas não tem acesso aos dados clínicos inseridos no app.

3. Permissões do Dispositivo

Para funcionar corretamente, o aplicativo solicita as seguintes permissões:

  • Câmera: Utilizada exclusivamente para a funcionalidade de "Scanner de QR Code", permitindo a importação de dados de pacientes compartilhados por outros colegas médicos. Nenhuma imagem ou vídeo é gravado ou transmitido.
  • Armazenamento: Necessário para salvar o banco de dados local e permitir a exportação/importação de backups manuais feitos pelo usuário.

4. Compartilhamento de Dados

O compartilhamento de informações clínicas ocorre apenas por ação deliberada do usuário através de:

  1. QR Code: O app gera um código contendo os dados do paciente para que outro dispositivo com o app instalado possa ler.
  2. Cópia de Texto/Compartilhamento Nativo: Ao usar a função de copiar evolução ou prescrição, os dados são enviados para a área de transferência do seu sistema operacional ou para o app que você selecionar (ex: WhatsApp, E-mail).

5. Segurança

Como os dados estão no seu dispositivo, a segurança deles depende também da segurança do seu aparelho (senha, biometria). Recomendamos o uso de proteções nativas do sistema operacional. No caso de exclusão do aplicativo, todos os dados locais de pacientes serão apagados permanentemente, a menos que você tenha realizado um backup manual.

6. Direitos do Usuário (LGPD)

Você tem total controle sobre os dados:

  • Acesso e Edição: Você pode visualizar e alterar qualquer dado a qualquer momento.
  • Exclusão: Você pode excluir registros individuais ou formatar o banco de dados do app nas configurações.
  • Portabilidade: Você pode exportar seus dados em formato JSON através da função de Backup.

7. Alterações nesta Política

Podemos atualizar esta política ocasionalmente. Recomendamos a revisão periódica desta página.
